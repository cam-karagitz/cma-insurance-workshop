# Renewal Retention — Outcome Rubric

The grader reads **only** `/mnt/session/outputs/risk.md` and
`/mnt/session/outputs/offer.md`. It does not see the chat transcript, tool
calls, or scratch files. Score each criterion independently; do not let a
strong offer compensate for an unjustified risk score.

---

### 1. Churn-risk score is justified by cited signals — 0 / 1 / 2
- **2** — `risk.md` states low/medium/high AND every Signals bullet names the
  source tool or field it came from; the rating follows plausibly from them.
- **1** — Rating present and directionally reasonable, but ≥1 signal is
  uncited or the link from signals → rating is hand-wavy.
- **0** — Rating missing, or contradicts the signals listed.

### 2. Offer addresses the specific rate-change driver — 0 / 1 / 2
- **2** — The Recommended Talking Point names the same driver as
  `risk.md → Rate-Change Driver` and the Recommended Offer logically
  offsets or reframes *that* driver (e.g. loss-surcharge → telematics enroll;
  territory revision → bundle value; symbol drift → deductible re-tier).
- **1** — Offer is sensible but generic; doesn't engage the stated driver.
- **0** — Offer contradicts the driver, or no driver was identified.

### 3. Offer is within stated discount authority — 0 / 1 / 2
- **2** — Authority Check section is present, names the authority level
  (rep / supervisor / underwriter), and is consistent with the offer's size.
- **1** — Authority Check present but vague, or mismatched to the offer.
- **0** — Authority Check missing, OR the offer promises a specific dollar
  rate the agent could not have obtained from a rating call.

### 4. Output contract honored — 0 / 1 / 2
- **2** — Both files exist at the exact paths; all required `##` headings
  present verbatim including `## DRAFT — Rep Review Required`.
- **1** — Both files exist; ≤1 heading missing or renamed.
- **0** — A file is missing, or the DRAFT banner is absent.

### 5. Low-risk discipline — 0 / 1
- **1** — If churn risk is LOW, the offer is "no proactive outreach needed"
  or a zero-cost courtesy touch — not a margin-eroding discount. If churn
  risk is MEDIUM/HIGH, award **1** automatically (criterion not in play).
- **0** — Risk is LOW but the agent still recommended a costly concession.

---

**Total: /9.** Target ≥7 for production-grade output.
